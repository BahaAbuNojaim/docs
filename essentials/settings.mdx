Configure your eZorro trading platform to align with your risk tolerance and investment objectives.

## Risk Management Settings

Set portfolio-level risk controls to protect your capital and manage exposure.

### Position Sizing

<ResponseField name="default_position_size" type="number" required>
Default position size as percentage of portfolio value.

Example: `0.02` (2% of portfolio per position)
</ResponseField>

<ResponseField name="max_position_size" type="number">
Maximum position size limit to prevent over-concentration.

Example: `0.10` (10% maximum per position)
</ResponseField>

### Portfolio Risk Controls

<ResponseField name="max_drawdown_limit" type="number">
Maximum portfolio drawdown before automatic position closure.

Example: `0.15` (15% maximum drawdown)
</ResponseField>

<ResponseField name="correlation_limit" type="number">
Maximum correlation between positions to ensure diversification.

Example: `0.7` (70% maximum correlation)
</ResponseField>

## Trading Parameters

### Execution Settings

<ResponseField name="slippage_tolerance" type="number">
Maximum acceptable slippage for order execution.

Example: `0.005` (0.5% maximum slippage)
</ResponseField>

<ResponseField name="order_timeout" type="integer">
Time in seconds before unfilled orders are cancelled.

Example: `300` (5 minutes)
</ResponseField>

### Strategy Parameters

<ResponseField name="lookback_period" type="integer">
Number of trading days for strategy calculations.

Example: `252` (1 trading year)
</ResponseField>

<ResponseField name="rebalance_frequency" type="string">
How often to rebalance positions.

Options: `daily`, `weekly`, `monthly`, `quarterly`
</ResponseField>

## Account Configuration

<CodeGroup>

```json Risk Settings
{
  "risk_management": {
    "max_portfolio_risk": 0.02,
    "position_sizing": {
      "default_size": 0.05,
      "max_size": 0.10,
      "volatility_adjusted": true
    },
    "stop_loss": {
      "default_percent": 0.05,
      "atr_multiple": 2.0
    }
  }
}
```

</CodeGroup>

### Advanced Risk Controls

<Accordion title="Correlation Monitoring">
<CodeGroup>

```json Correlation Limits
{
  "correlation_controls": {
    "max_pairwise_correlation": 0.7,
    "lookback_period": 60,
    "rebalance_on_breach": true,
    "alert_threshold": 0.65
  }
}
```

</CodeGroup>

The platform monitors correlations between positions and automatically rebalances when limits are exceeded.
</Accordion>

## Broker Integration

Connect your trading account to execute strategies:

<ResponseField name="broker" type="string" required>
  Supported brokers: Interactive Brokers, TD Ameritrade, Alpaca, Schwab
</ResponseField>

<ResponseField name="account_type" type="string">
  Account type: `margin`, `cash`, or `retirement`
</ResponseField>

<ResponseField name="api_credentials" type="object">
  Secure API credentials for broker connection
  
  <Expandable>
    <ResponseField name="api_key" type="string" required>
      Broker API key (encrypted at rest)
    </ResponseField>
    
    <ResponseField name="secret_key" type="string" required>
      Broker secret key (encrypted at rest)
    </ResponseField>
  </Expandable>
</ResponseField>

## Market Data Configuration

Configure data feeds for strategy execution:

<CodeGroup>

```json Data Sources
{
  "market_data": {
    "primary_source": "Interactive Brokers",
    "backup_source": "Alpha Vantage",
    "real_time": true,
    "after_hours": true,
    "fundamental_data": true
  }
}
```

</CodeGroup>

<Tip>
Real-time data requires a market data subscription. Start with delayed data for strategy development and testing.
</Tip>

## Trading Notifications

Configure alerts and notifications for trading events:

<ResponseField name="alerts" type="object">
  Trading alert configuration
  
  <Expandable>
    <ResponseField name="order_fills" type="boolean" default="true">
      Notify when orders are filled
    </ResponseField>
    
    <ResponseField name="stop_loss_triggers" type="boolean" default="true">
      Alert when stop losses are hit
    </ResponseField>
    
    <ResponseField name="drawdown_warnings" type="boolean" default="true">
      Warn when approaching maximum drawdown
    </ResponseField>
    
    <ResponseField name="correlation_breaches" type="boolean" default="true">
      Alert when correlation limits are exceeded
    </ResponseField>
  </Expandable>
</ResponseField>

## Performance Monitoring

Configure portfolio performance tracking:

<CodeGroup>

```json Performance Settings
{
  "performance_tracking": {
    "benchmark": "SPY",
    "rebalance_costs": 0.005,
    "risk_free_rate": 0.03,
    "lookback_periods": {
      "sharpe_calculation": 252,
      "volatility_window": 30,
      "correlation_window": 60
    }
  }
}
```

</CodeGroup>

<Warning>
**Important**: Changes to risk settings take effect immediately for new positions. Existing positions will be adjusted at the next rebalancing interval.
</Warning>
