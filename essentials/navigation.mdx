Monitor and control your live trading algorithms:

```python Strategy Status Display
# Example of strategy status information shown in the interface
```

<Accordion title="Strategy Controls">
**Available Actions:**
- **Pause Strategy**: Stop new position entries
- **Resume Strategy**: Continue normal operation  
- **Modify Parameters**: Adjust risk settings or signals
- **View Performance**: Detailed analytics and charts
- **Export Data**: Download trade history and metrics
</Accordion>

### Strategy Builder Navigation

<Steps>
<Step title="AI Assistant Access">
Click the **"Create Strategy"** button to launch the natural language interface:

- Describe your trading idea in plain English
- Review AI-generated algorithm logic
- Customize parameters and risk controls
- Run initial backtests
</Step>

<Step title="Parameter Optimization">
Navigate to **Strategy → Optimize** to fine-tune your algorithm:

- Define parameter ranges for testing
- Select optimization metrics (Sharpe, return, drawdown)
- Run walk-forward analysis
- Review out-of-sample results
</Step>

<Step title="Deployment Workflow">
Move from development to live trading:

1. **Backtest Results** → Validate historical performance
2. **Paper Trading** → Test with live data, zero risk
3. **Live Deployment** → Allocate real capital
4. **Monitor Performance** → Track and adjust as needed
</Step>
</Steps>

### Account Management

<Accordion title="Broker Integration Setup">
**Navigation**: Settings → Accounts → Add Broker

**Supported Brokers:**
- Interactive Brokers (IBKR)
- TD Ameritrade  
- Alpaca Markets
- Charles Schwab
- E*TRADE

**Required Information:**
- Account number
- API credentials (key and secret)
- Account type (cash, margin, retirement)
- Trading permissions verification
</Accordion>

<Accordion title="Data Subscriptions">
**Navigation**: Settings → Market Data → Subscriptions

**Available Data Feeds:**
- **Level I**: Real-time quotes and trades
- **Level II**: Order book depth
- **Historical**: Minute, daily, and tick data
- **Fundamental**: Earnings, financials, estimates
- **Options**: Chains, Greeks, and flow data

**Pricing Tiers:**
- **Free**: 15-minute delayed data
- **Professional**: Real-time US equities ($25/month)
- **Premium**: Multi-asset real-time ($75/month)
</Accordion>
